# ✨ SimStory

> **Build a Sim you didn't plan.**

SimStory is a browser-based randomizer for **The Sims 4** — pick your owned packs, choose a life stage, and instantly generate a fully valid character (or an entire household) ready to play.

🎮 **[Live Demo →](https://pixelplumbob.github.io/sims-randomizer/)**

---

## 🌟 Features

| | |
|---|---|
| 🎲 **Smart randomization** | Generates traits, aspirations, life states, worlds, careers, and more |
| 📦 **Pack-aware** | Only rolls content you actually own — supports all packs up to *Royalty & Legacy*, *Adventure Awaits* & *Enchanted by Nature* |
| 🚫 **Conflict-free traits** | Built-in graph of 80+ incompatible trait pairs (e.g. *Evil + Good*, *Lazy + Neat*) — broken combos are impossible |
| 🧒 **Life stage logic** | Correct career pools, trait counts and restrictions across all stages: `Infant → Toddler → Child → Teen → Young Adult → Adult → Elder` |
| 👻 **Ghost system** | 10% chance your Sim is a ghost, with cause-of-death filtered by life state (e.g. only Vampires can die of Sunlight) |
| 🧬 **Occult inheritance** | Children are more likely to share their parents' life state |
| 🔒 **Lock sections** | Pin any part of your Sim and re-roll everything else |
| ⛔ **Exclude items** | Blacklist specific traits, aspirations, careers or life states from ever appearing |
| 👨‍👩‍👧 **Family generation** | Auto-prompts to generate a spouse and kids with age-appropriate life stages |
| 📓 **Save as Notebook** | Export your entire household to a standalone, printable HTML file |
